## 표현식
- 어떠한 값을 만들어 내는 간단한 코드

---

## 값
- 표현식이 평가된 결과

---

## 평가
- 표현식을 계산하여 그 결과인 값을 만들어내느 과정

---

## 문장
- 표현식이 하나 이상 모인 것

---

## 프로그램
- 문제를 해결하기 위한 명령어들의 집합

---

## 키워드
- 특별한 의미가 부여된 단어
- 파이썬이 만들어질 때 이미 사용하겠다고 예약해 놓는 것
- 사용자가 이름을 정할 때 키워드 사용하면 안됨

---

## 식별자
- 프로그래밍 언어에서 이름을 붙일 때 사용하는 단어
- 변수 또는 함수 이름 등으로 사용됨
  - 키워드 사용 X
  - 특수 문자는 언더바만 허용
  - 숫자로 시작 X
  - 공백 포함 X

---

## 주석
- 프로그램의 진행에 전혀 영향을 주지 않는 코드
- 프로그램을 설명하기 위해 사용
- `#` 기호를 붙여 주석 처리

---

## 변수
- 값을 나중에 다시 사용하기 위해, 그 값에 붙여주는 고유한 이름

---

## 변수 할당
- 표현식이 만들어 낸 값에 이름을 붙이는 과정

---

## 재할당
- 이미 값이 할당된 변수에 새로운 값을 다시 할당 하는 것

---

## 타입
- 변수나 값이 가질 수 있는 데이터의 종류

---

## 데이터 타입
- 값의 종류와 그 값으로 할 수 있는 '동작(연산)'을 결정하는 속성
- 데이터 타입을 확인할 때는 type()함수를 사용

---

## 데이터 타입 분류 5가지
- Numeric Types
- Text Sequence Type
- Sequence Types
- Non-Sequence Types
- Boolean, None, Functions

---

## 정수형 데이터 (Numeric Types)
- 프로그래밍에서 값을 계산하고, 수량을 세고, 데이터를 분석하는등 가장 기본이 되는 데이터 타입
  - 정수형(int)
  - 실수형(float)

---

## 정수 자료형 (int)
- 소수점이 없는 숫자를 표현(양수, 음수, 0)

---

## 실수 자료형 (float)
- 소수점이 있는 더 정밀한 숫자를 표현

---

## 지수 표현법
- 아주 크거나 아주 작은 실수를 간결하게 표현하기 위해 사용하는 방식
- 'e' 또는 'E'를 사용
```python
# 1,000,000,000 (1 * 10^9)
big_number = 1e9

# 0.00123 (1.23 * 10^-3)
small_number = 1.23e-3
```

## 문자열 (str)
- 문자들의 순서가 있는 변경 불가능한 시퀀스 자료형

---

## 문자열 만들기
- 작은따옴표 또는 큰 따옴표로 감싸서 표현
```python
# 하나만 출력합니다.
print('하나만 출력합니다.')
print('hello world')

# 여러개를 출력합니다.
print('여러개를 출력합니다.')
print(12, 13, 14, 15)
print('안녕', '하세요', '반갑', '습니다.')
```

---

## 문자열 내부에 따옴표 넣기
- 큰 따옴표를 문자열 내부에 넣고 싶은 경우
```python
print('"안녕하세요"라고 말했습니다.')
```

- 작은 따옴표를 문자열 내부에 넣고 싶은 경우
```python
print("'안녕'이라고 말했습니다.")
```

---

## 이스케이프 문자를 사용해 문자열 만들기
- 역슬래시와 문자를 조합해 특별한 기능을 수행
- \와 함께 작은따옴표, 큰 따옴표를 사용하면 이를 '단순한 따옴표'로 인식
```python
print("\"hi\"hello")
print('\'hello\'hi')
```

---

## 이스케이프 시퀀스 예약 문자 정리
|예약 문자|기능
|:-:|:-:
|\n|줄 바꿈
|\t|탭
|`\\`|백슬래시
|\'|작은 따옴표
|\"|큰 따옴표

---

## 여러 줄 문자열 만들기
- 큰 따옴표 또는 작은 따옴표를 세 번 반복한 기호를 사용
```python
print('''
동해물과 백두산이 마르고 닳도록
하느님이 보우하사 우리나라 만세
무궁화 삼천리 화려강산 대한사람
대한으로 길이 보전하세
''')
```
`이렇게 코드를 작성하면 첫 번째 줄과 마지막 줄에 의도하지 않은 줄바꿈이 들어가게 된다.  
만약 줄바꿈을 하지 않고 코드를 입력하고 싶다면 \ 기호를 사용`

## f-string
- 문자열 내에 변수나 표현식의 결과를 손쉽게 삽입하는 방법
- 문자열 시작 전 **f** 접두어를 붙이고, 삽입할 부분을 중괄호로 감싸줌

```python
name = '김철수'
age = 30

introduce = f'안녕, 내 이름은 {name}이고 나이는 {age}살 입니다.'

print(introduce)
```

## 시퀀스로서의 문자열 활용
- 문자열은 시퀀스이므로 인덱싱, 슬라이싱, 길이 확인, 반복 등 공통 기능을 모두 사용할 수 있음

## 인덱스 (index)
- 시퀀스 자료형에ㅐ서 각 값의 위치를 식별하기 위해 부여된 고유한 번호
- 인덱스 번호는 0부터 시작
  - 시작점으로 부터 떨어진 거리를 의미 따라서 첫번째 값은 시작점 임으로 0을 의미
- 파이썬은 음수 인덱스를 지원
  - -1은 맨 마지막 값을 의미

---

## 슬라이싱 (slicing)
- 시퀀스의 일부분을 잘라내어 새로운 시퀀스를 만드는 작업

---

## 슬라이싱 사용 방법
- 대괄호 안에 시작 위치, 끝 위치, 간격을 콜론으로 구분하여 지정
```python
sequence[strat:stop:step]
```
1. start: 슬라이싱을 시작할 인덱스
2. stop: 슬라이싱을 끝낼 인덱스 (포함되지 않음)
3. step: 몇 개씩 건너뛰며 값을 가져올지에 대한 간격

---

## 진법 표현하기
- 파이썬은 코드 내에서 다양한 진법의 숫자를 직접 표현할 수 있도록 특별한 접두사를 제공  

|진법|접두사|사용하는 숫자/문자
|:-:|:-:|:-:
|2진수 (binary)| 0b|0과1
|8진수 (octal)|0o|0 부터 7 까지
|16진수 (hexadecimal)|0x|0부터 9, a 부터 f까지
