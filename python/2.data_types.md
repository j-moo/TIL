## 타입 (Type)
- 변수나 값이 가질 수 있는 데이터의 종류를 의미
> 어떤 종류의 데이터인지, 어떻게 해석되고 처리되어야 하는지를 정의

---

## 타입의 구성 요소
- 타입은 2가지 요소로 구분 **값**과 **값에 적용할 수 있는 연산**
- 값은 다양한 타입이 존재 따라서 **값이 가질 수 있는 여러 종류를 구분하기 위해 타입이 필요**

---

## Data Types
- 값의 종류와 그 값으로 할 수 있는 '동작(연산)'을 결정하는 속성

---

## Data type이 필요한 이유
- 각 타입에 따라 가능한 기능과 연산이 다르기 때문
  - 숫자는 더하기 가능 : 1+2=3
  - 문자열은 결합 가능 : "안" + "녕" = "안녕"
  - 숫자와 문자는 서로 더할 수 없음

---

## Data type 분류 5가지
- Numeric Type
  - int(정수), float(실수), complex(복소수)
- Text Sequence Type
  - str(문자열)
- Sequence Type
  - list, tuple, range
- Non-sequence Types
  - set, dict
- 기타
  - Boolean, None, Functions
